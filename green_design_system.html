Guía de Uso

✓ Hacer

  • • Usar green-800 como color principal de marca
  • • Aplicar emerald para estados de éxito
  • • Usar slate-900 para títulos
  • • Mantener fondos blancos o gray-50
  • • Usar bordes slate-200 para separar
  • • Tab activo: green-800 con fondo o borde
  • • Tabs inactivos: gray-600 con hover
  • • Gráficos: usar colores pastel (300) para visualizaciones
  • • Combinar máximo 5-6 colores pasteles en un gráfico

✗ Evitar

  • • No mezclar más de 2 verdes juntos
  • • Evitar lime en grandes áreas
  • • No usar green-800 en texto largo
  • • Evitar fondos muy oscuros
  • • No sobrecargar con colores
